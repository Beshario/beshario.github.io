A mashup web app providing an interactive guide to BART (Bay Area Rapid Transit) routes, built as part of an audited Harvard course, [Building Dynamic Websites](http://cs75.tv/2012/summer/) by David J. Malan.

- Routes drawn as polylines on Google Maps in official BART route colors
- Station markers with real-time departure and arrival info on click
- Backend built with PHP, jQuery, Ajax (XML and JSON)
- Third-party APIs: Google Maps and BART API
- SQL (MySQL) used for web caching
