YesCoach is a live web application for planning, tracking, and progressing through personalized workout programs. Started as a hobby project in 2017 to explore web technologies. [Available here](https://yes-coach.web.app/), best viewed in mobile.

### What It Does

Users create custom workout programs, log sessions, and track progress over time through a mobile-first interface.

### How It Was Built

- UX and graphic design in Adobe XD
- Frontend built with Angular and Ionic (HTML, CSS, JavaScript)
- Google Cloud Platform for data management and behavior analytics
- Auxiliary CRUD app for managing exercise content in the database
